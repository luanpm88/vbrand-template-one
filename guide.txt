
setup
    1. Cài Woocomerce và cấu hình: để sinh ra các page cần thiết: Shop, cart, ..
    2. Vào thư mục themes của wordpress: cd wp-content/themes/
    3. Clone vBrand Theme One: git clone git@github.com:canhoduclong/vbrand-template-one.git
    4. Đăng nhập vào WordPress Admin Panel, vào menu Appearance, activate theme vBrand Template One
    5. Vào menu Pages, tạo 1 mới 1 page bất kỳ (VD: Homepage). Bên tay phải mục Template, chọn Shop page define. Chọn publish
    6. Vào menu Appearence => Customeize => Home sitting => HomePage => Home page ( chọn Home page). Chọn save
